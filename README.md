# E-Student Portal Project Page

This page provides detailed information about the **E-Student Portal** project, developed by **Mustafa Obiedat**. The project is designed to help students of **Hashemite University** with course registration and academic schedule management.

## Navigation Bar
The navigation bar at the top of the page allows easy access to:
- **Home**
- **Projects**
- **About**

## Project Overview

### Project Name
**E-Student Portal**

### Project Description
This platform allows **Hashemite University students** to register for courses and manage their academic schedules.

### Project Image
A screenshot of the portal interface is displayed here, showing the design of the E-Student Portal.



### About the Project
This project involves developing the backend of the E-Student Portal using **Node.js** and **Express.js** to handle student registrations, payments, and other essential functions. The portal uses **MongoDB** for storing student data and includes secure user authentication using **Bcrypt** for password protection.

### Challenges
Some of the main challenges in developing this application included:
- Ensuring scalability during peak registration periods
- Maintaining data integrity and preventing overbooking of courses
- Implementing secure authentication and protecting sensitive data
- Integrating a payment gateway and optimizing backend performance

## Results
The results section highlights the portal's user interface with images of the **Login Page** and **Dashboard Page**.

## Footer
The footer includes:
- **Navigation links** to Home, Projects, About, and Contact pages
- A **Call-to-Action (CTA)** for getting in touch or browsing more projects
- **Social media links** to Instagram, GitHub, and LinkedIn

### Social Media Links:
- Instagram
- [GitHub](https://github.com/mostafa-obiedat)
- [LinkedIn](https://www.linkedin.com/in/mustafa-obiedat/)

## Design and Layout
The layout is designed to be clean and responsive. It includes:
- A **responsive navigation bar** and project information section
- **Project images** to showcase the portal interface
- A **sticky sidebar** for project details (client info, year, and role)
- A **flexbox layout** for better structure and alignment

### Mobile Responsiveness
The page is fully responsive, adjusting the layout for different screen sizes using **media queries**. For example:
- For smaller screens, the project description and sidebar are rearranged to ensure readability and usability.
- **Images** are resized and optimized for mobile viewing.

## Technologies Used
- **Frontend**: HTML, CSS
- **Backend**: Node.js, Express.js, MongoDB
- **Authentication**: Bcrypt encryption for secure password handling
